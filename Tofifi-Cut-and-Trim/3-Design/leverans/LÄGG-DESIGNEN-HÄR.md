# Lägg designen här

När Claude Design är klar — spara filerna den levererar **i den här mappen**:

```
3-Design/leverans/
├── index.html
├── css/...
├── js/...
└── assets/...
```

Goal 02 (`2-Byggplan/02-Goal-design-och-innehåll.md`) hämtar designen härifrån
och flyttar in den i `5-Kod/`.

Är den här mappen tom när Goal 02 körs — stoppar Claude Code och rapporterar
att designen saknas. Kör då Claude Design först.

(Den här filen kan tas bort när designen är på plats.)
